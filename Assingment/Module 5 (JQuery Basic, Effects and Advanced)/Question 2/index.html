How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

1. CSS USING jQuery.

jQuery(document).ready(function(){
jQuery(".q1").css({"width":"50%","height":"auto","background-color":"rgb(212, 158, 237)","padding":"20px", "margin":"10px" , "border":"2px solid"});
});

2. ADDING CLASS USING jQuery.

jQuery(document).ready(function(){
jQuery(".q2").addClass("header");
});

3. REMOVNG CLASS USING jQuery.

jQuery(document).ready(function(){
jQuery("#btn").click(function(){
jQuery(".q3").remove(); });
});
