Question 8 : In Bootstrap 4, what is flexbox?

The .d-flex class is applied to the container to create a flex container. This activates the Flexbox behavior.

The .p-2 class applies padding to each flex item.

The .flex-fill class ensures that the flex items share the available space equally.
