Question 9 : How can one create an alert in Bootstrap?

-> Each alert is created using the alert class along with a specific contextual class (alert-success, alert-info, alert-warning, alert-danger) to indicate the type of alert.

-> The role="alert" attribute is added for accessibility purposes.

-> The alert content is placed within the div container, providing the message or information you want to display.
