Question 2 : What is the use of isNaN function?

The isNaN() function in JavaScript is used to determine whether a value is "Not-a-Number" (NaN). It takes a single argument and returns a boolean value indicating whether the argument is NaN or not.

If the argument is not a number, or it cannot be converted to a number, the function returns `true`. This includes values like NaN, strings that cannot be parsed as numbers, and non-numeric values.

If the argument is a valid number or a string representing a numeric value, the function returns `false`.

ex:
console.log(isNaN(42)); // Output: false
console.log(isNaN('42')); // Output: false
console.log(isNaN('Hello')); // Output: true
console.log(isNaN(NaN)); // Output: true
console.log(isNaN(true)); // Output: false (1 is considered a number)
console.log(isNaN(undefined)); // Output: true
